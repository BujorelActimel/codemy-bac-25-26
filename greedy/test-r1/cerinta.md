1. Se dau trei numere naturale n, a și b, 1 ≤ a < b < n.
   Să se determine o modalitate de a-l scrie pe n ca sumă de termeni egali cu a sau b în care numărul
   de termeni egali cu a este maxim.
   Exemplu
   40 3 5 => 3 3 3 3 3 3 3 3 3 3 5 5
2. Se dau n numere naturale.
   Să se determine o subsecvență (formată din elemente alăturate) astfel încât suma elementelor din
   această subsecvență să fie divizibilă cu n.
   Se vor afișa indicii care determină această subsecvență.
   În cazul în care există mai multe soluții se poate afișa oricare dintre acestea.
   Exemplu
   7
   65 26877 13488 43 256888 53 428
   =>
   3 4
   Rezolvările implementate vor fi bazate pe utilizarea metodei greedy.
